# Neurotechnology Project README

## Overview
This project focuses on two key areas:

1. **P300 EEG Speller (at Pz)**: A brain-computer interface (BCI) that uses the P300 event-related potential (ERP) detected via EEG at the Pz electrode location to control a speller system. This system allows users to spell characters using their brain waves by focusing on specific targets in a grid.

2. **Price Sensitivity using ECG and GSR**: This component analyzes the correlation between consumer price sensitivity and physiological responses (ECG and GSR). The objective is to determine how changes in price influence emotional and physiological reactions, helping understand consumer behavior better.
